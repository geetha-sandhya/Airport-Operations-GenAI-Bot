# ✈️ Airport Ground Operations Explainer Bot

## 📌 Project Overview
This project is a Generative AI-based chatbot developed during a college hackathon conducted by V Cube Software Solutions.

The chatbot helps passengers understand airport ground operations and passenger flow in a simple and clear way.

---

## ❗ Problem Statement
Passengers often face confusion during airport processes such as:
- Check-in
- Security screening
- Boarding
- Arrival and baggage claim

Due to:
- Lack of clear guidance
- Heavy crowd during peak hours
- Limited helpdesk availability

---

## 💡 Solution
To solve this, we developed an AI-powered chatbot that:
- Provides step-by-step explanations of airport procedures
- Reduces passenger confusion
- Improves overall airport experience

---

## 🛠️ Tech Stack
- **Python** – Backend logic
- **Streamlit** – Web interface
- **Google Gemini API** – Generative AI responses
- **Prompt Engineering** – Controlled and safe AI output

---

## ⚙️ Features
- Explains only airport-related procedures
- Filters unrelated questions
- Uses system prompts for safe responses
- Simple and interactive UI

---

## 🧠 How It Works
1. User enters a query related to airport procedures  
2. Input is filtered to allow only relevant topics  
3. Query is sent to Google Gemini model  
4. AI generates a clear and safe response  
5. Response is displayed in the Streamlit web app  

---

## 📊 Impact
- Reduces passenger confusion and stress  
- Saves helpdesk time by automating common queries  
- Improves passenger flow and efficiency

## ▶️ How to Run the Project

### Step 1: Install dependencies
pip install -r requirements.txt

### Step 2: Run the application
streamlit run airportbot.py

### Step 3: Open in browser
The app will open at:http://localhost:8501

---

## 🔐 Important Note
- You need a **Google AI Studio API Key** to run this project  
- Enter the API key in the sidebar when running the app  
- API key is not stored for security reasons
